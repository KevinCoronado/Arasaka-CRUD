# Arasaka Corp School

Aplicación web desarrollada como sistema escolar donde los profesores pueden iniciar sesión y gestionar alumnos y sus calificaciones.  
El sistema permite agregar, editar y eliminar alumnos, así como registrar, modificar y eliminar sus calificaciones.

---

## Tecnologías utilizadas

- Frontend: React + Chakra UI  
- Backend: .NET (C#)  
- ORM: Entity Framework Core  
- Base de datos: MySQL  
- Comunicación Front/Back: API REST  
- Estilos: Chakra UI  

---

## Funcionalidades principales

- Login de profesor  
- Gestión de alumnos: crear, editar, eliminar, listar  
- Gestión de calificaciones: agregar, editar y eliminar notas  
- Persistencia de datos mediante MySQL y Entity Framework  
- Interfaz moderna y responsive con Chakra UI
